    # VuePress上传github无权限问题

    # 检查

    1. 检查有没有在GitHub的https://github.com/settings/keys上添加你本机的SSH key
    2. 用户主目录下，看看有没有.ssh目录 .ssh目录下有没有id_rsa和id_rsa.pub这两个文件
    C:\Users\Administrator\.ssh

    # 设置秘钥

    1. Windows下打开Git Bash(C:\Windows) .一路回车，使用默认值即可。
    $ ssh-keygen -t rsa -C "youremail@example.com"
    1. 登陆GitHub，打开“Account settings”，“SSH Keys”页面： 然后，点“Add SSH Key”，填上任意Title，在Key文本框里粘贴id_rsa.pub文件的内容
