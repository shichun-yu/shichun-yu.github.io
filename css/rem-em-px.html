# CSS 常用的单位

# rem

# rem原理

  • 常识: 浏览器默认的字体高都是16px
  • 兼容性: IE9+, Firefox, Chrome, Safari, Opera的主流版本都支持rem
  • 使用 % 单位方便使用
// 设置html的 font-size= 62.5%
// 因为 100% = 16px； 1px = 6.25%； 所以10px = 62.5%
// 得出结果  1rem = 10px,   px与rem的转换通过10

# demo

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>rem</title>
  <style>
    * {
      margin: 0;
      padding: 0;
    }

    html {
      font-size: 62.5%;
    }

    .test {
      width: 39.5rem;
      height: 20rem;
      background-color: bisque;
      text-align: text；
    }

    .hello {
      color: red;
    }
  </style>
</head>

<body>
  <div class="test">
    <p class="hello">我是小石</p>
  </div>
</body>

</html>
